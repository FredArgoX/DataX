Data Science Questions

What is Machine Learning?

Machine Learning (ML) is a subset of Artificial Intelligence (AI), which focuses on building systems capable of learning patterns from data, to perform specific tasks, like predicting or making decisions. As more data is fed into an ML system, the system's performance can increase.

What is Supervised Learning?

Supervised Learning is a type of ML, where a model learns patterns from labeled data, i.e., the model learns the relationship between input-output pairs. Once the model is trained, it is capable of predicting an output for each input given.

What is Unsupervised Learning?

Unsupervised Learning is a type of ML, where a model learns patterns from unlabeled data, i.e., as the data is not structured as input-output pairs, but only inputs, the model tries to find patterns or groupings within the data itself.

What is Semi-Supervised Learning?

Semi-Supervised Learning is a type of ML, that combines supervised and unsupervised learning. A model learns patterns from labeled data, generating pseudolabels from the unlabeled data, which are then used to retrain the model, hopefully increasing its performance.

What is Reinforcement Learning?

Reinforcement Learning (RL) is a type of ML. It was inspired by how humans and animals learn through trial and error. An agent interacts with an environment by executing specific actions. The agent gets feedback in the form of rewards or penalties based on the consequences of the actions executed. The goal of the agent is to learn a policy that maximizes the total reward.

What is Deep Learning?

Deep Learning is a subfield of ML. It is inspired by how the human brain works, by utilizing Artificial Neural Networks (ANNs) (typically with many layers, hence 'deep') to automatically learn relevant features from raw data. In order to train highly efficient ANNs, large datasets and high computational power (like GPUs or TPUs) are required.

What is Ensemble Learning?

Ensemble Learning is an ML technique focused on combining multiple individual models (often called base learners, weak learners, base models, or learners) to produce a more robust model, i.e., a model with lower error rates, reduced overfitting, and improved generalization (capable of capturing a wider range of patterns in the data).

What is Meta-Learning?

While traditional ML produces models specialized in the execution of a single task, Meta Learning, also known as 'learning to learn' aims to develop models capable of rapidly adapting to perform new tasks with limited data.

A meta-learning model is trained across different, but related, tasks, from which the model learns commonalities between them, extracting meta-knowledge about the learning process itself, that enables rapid adaptation to new tasks.

The trained model can then be employed to perform a new, but related, task, with minimal data or training.
