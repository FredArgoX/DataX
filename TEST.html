About This Concept Map

This is a modular, responsive, and fully interactive concept map designed to illustrate the core branches of Machine Learning. The structure is built using scalable vector graphics (SVG) to maintain crisp visuals and high performance across devices.

You can interact with the dropdown menu above to highlight specific branches such as Supervised, Unsupervised, or Reinforcement Learning. Hovering over nodes gives visual feedback, and future versions may include clickable nodes that open detailed descriptions.
