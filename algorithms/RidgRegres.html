Ridge Regression

Supervised Learning

Regression

Ridge Regression is a regularized extension of linear regression designed to improve performance in scenarios with multicollinearity, high-dimensional datasets (many more features than observations), when the OLS model overfits, or when there's a need to stabilize model coefficients without eliminating any predictors (unlike Lasso). It adds a penalty term to the loss function that discourages the model from fitting large coefficient values, reducing the risk of overfitting.

Ridge Regression modifies the ordinary least squares (OLS) objective by adding an L2 penalty term to the cost function:

$$\text{Loss} = \sum (y_i - \hat{y}_i)^2 + \lambda \sum \beta_j^2$$

$ \sum (y_i - \hat{y}_i)^2$ represents the sum of squared residuals.

$\lambda \sum \beta_j^2$ is the L2 penalty.

$\lambda$ (lambda) is a hyperparameter (called alpha in scikit-learn) that controls the strength of the penalty.

To perform a successful Ridge Regression analysis, the use of numerical features must be ensured (categorical feature encoding). Feature scaling (standarization) is required, as Ridge is sensitive to the scale of the features.

While Ridge handles multicollinearity, the assumptions that still must hold true are linearity, homoscedasticity, independence, and normality of residuals,
