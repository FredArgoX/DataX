Na誰ve Bayes is a probabilistic classification algorithm based on Bayes' Theorem. It calculates the posterior probability of a class given the observed features and classifies data points by choosing the class with the highest probability.

It assumes that the features in the dataset are independent of each other (this is the "na誰ve" assumption). It also assumes that all features contribute equally to the outcome.

Unlike many machine learning algorithms (such as K-Nearest Neighbors, Logistic Regression, or Support Vector Machines), Na誰ve Bayes does not require feature scaling because it is based on probability calculations from the data's distribution rather than distance-based metrics.

If using GaussianNB, categorical variables need to be converted to numerical form (encoding like one-hot encoding or label encoding) and standardization may improve numerical stability if feature magnitudes vary significantly, but other NB methods can directly work with categorical data, CategoricalNB (best for categorical features), BernoulliNB (for binary categorical features), MultinomialNB (for count-based categorical data).
